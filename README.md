# Langfarm

Langfarm 是 LLM 应用的 DevOps 平台。使用 Langfuse 及对其改进等...

## 特性

* Tracing：
  * tracing 数据（traces 和 observations）转发到 kafka。
  * tracing 数据存储在 Paimon 数据湖。
